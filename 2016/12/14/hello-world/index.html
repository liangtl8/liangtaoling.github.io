    第二次发布

    • 每次有更改后必须要重新生成页面发布

    如果想引用别人的模板打开https://hexo.io 在里面找自己想要的 点击蓝色标题 打开之后复制地址 打开命令行 git clone +刚才复制的地址
    克隆完后 在文件里打开themes 文件夹 把克隆的文件放到这个文件夹里 然后打开_config.yml 找theme 把后面的模板名称改成你下载好这个
    在去命令行执行 hexo g -> hexo deploy
